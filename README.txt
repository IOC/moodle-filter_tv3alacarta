TV3 A LA CARTA FILTER PLUGIN
============================

TV3 a la carta is a Moodle 2.X filter that replace any TV3 a la carta link with an embedded player

Web: https://github.com/IOC/moodle-filter-tv3alacarta


Install
=======

Download code from gitHub, and copy inside directory MOODLE_PATH/filter/tv3alacarta
